UI Kit — документация

На странице представлены базовые компоненты интерфейса и примеры их использования. Именование выполнено по БЭМ: блок, блок__элемент, блок--модификатор.

Кнопки

Блок .btn. Модификаторы: .btn--primary, .btn--secondary, .btn--disabled (или атрибут disabled); размеры: .btn--sm, .btn--md, .btn--lg. Поддерживаются состояния hover/active/focus.

<button class="btn btn--primary btn--md">Primary</button>
<button class="btn btn--secondary btn--sm">Secondary sm</button>
<button class="btn btn--lg" disabled>Disabled</button>

Поля ввода

Обёртка поля — блок .field с элементами .field__label, .field__message. Сам инпут — .input; модификатор ошибки — .input--error. Для иконки используйте обёртку .input-wrapper и элемент .input__icon, а инпуту добавьте .input--with-icon.

<label class="field">
  <span class="field__label">Поиск</span>
  <div class="input-wrapper">
    <svg class="input__icon" ...></svg>
    <input class="input input--with-icon" type="text" placeholder="Найти...">
  </div>
</label>

Dropdown

Блок .dropdown; элементы: .dropdown__toggle, .dropdown__menu, .dropdown__item. Состояние раскрытия задаётся классом .is-open на корневом элементе (управляется JS).

Пункт 1 Пункт 2 Пункт 3 
<div class="dropdown">
  <button class="dropdown__toggle btn btn--primary" data-dropdown-toggle>Открыть меню</button>
  <div class="dropdown__menu">
    <a class="dropdown__item" href="#">Пункт 1</a>
  </div>
</div>

Чекбоксы и радиокнопки

Общий блок управления — .control. Элементы: скрытый инпут .control__input, визуальная коробка .control__box и отметка .control__dot, подпись .control__label. Для радио используйте модификатор блока .control--radio.

<label class="control">
  <input class="control__input" type="checkbox">
  <span class="control__box"><span class="control__dot"></span></span>
  <span class="control__label">Согласен</span>
</label>

Карточки

Блок .card с элементами .card__img, .card__body, .card__title, .card__text. Модификаторы: .card--no-image (без изображения), размеры .card--sm, .card--md, .card--lg.

Маленькая карточка

Размер small. Подходит для компактных галерей или списков.

Средняя карточка

Размер medium. Сбалансированный вариант для большинства страниц.

Большая карточка без изображения

Модификатор .card--no-image скрывает изображение. Размер large.

<article class="card card--md">
  <img class="card__img" src="..." alt="" />
  <div class="card__body">
    <h3 class="card__title">Заголовок</h3>
    <p class="card__text">Описание...</p>
  </div>
</article>

Сетки и контейнер

Контейнер: класс .container центрирует контент и ограничивает ширину. Grid: используйте комбинации классов .grid + модификаторы колонок (.grid--cols-2, .grid--cols-3, .grid--cols-4) и респонсив-модификаторы (.grid--md-cols-3, .grid--md-cols-4).

<div class="grid grid--cols-2 grid--md-cols-4">
  <div>...</div>
</div>

Методология БЭМ

Блок — независимый компонент (напр., .btn, .card).

Элемент — часть блока через __ (напр., .card__title).

Модификатор — вариант через -- (напр., .btn--primary, .card--lg).
